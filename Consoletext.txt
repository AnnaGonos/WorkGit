Команды     Предназначение                                                     Пример
git init    Превращает обычный каталог в Git репозиторий                       $  git init
git clean   Используется для удаления мусора из рабочего                       $ git clean -f -d
git merge   Используется для слияния одной или нескольких веток в текущую      $ git merge hotfix (находясь в ветке master)
